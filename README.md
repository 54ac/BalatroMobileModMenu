## Balatro Mobile Mod Menu

### What is this?

This is a mod for the mobile version of Balatro which adds a tab in the settings menu with various options:

- Vibration: Fixes and enables haptics on Android,
- Readable font: Replaces the pixelated font with a sans-serif font for better readability,
- FPS cap: Adds selectable FPS limits (30, 45, 60, 90, 120),
- Skip tutorial: Skips the tutorial for new profiles,
- Censored collab names: Restores original face-card set collaboration names which are censored on Android.

### Usage

- [Download this repository as a ZIP file](https://github.com/54ac/BalatroMobileModMenu/archive/refs/heads/main.zip).
- Use the [Balatro APK Injector](https://github.com/54ac/BalatroAPKInjector) to add Lovely to your legally acquired copy of Balatro.
- Go to the data folder of Balatro (e.g. `Android/data/com.playstack.balatro.android.mod`).
- Extract the contents of this repository into the `files/mods` folder, so that the folder structure looks like this: `files/mods/BalatroMobileModMenu-main/lovely`.

The additional tab should now appear in the settings menu.
